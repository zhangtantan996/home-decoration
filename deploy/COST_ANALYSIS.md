## 2. 阿里云避坑指南 (Tips)

### 2.1 实例选型
*   ✅ **推荐**: **u1 (通用算力型)**, **e (经济型)**, **c7/g7 (计算型)**。
*   ❌ **不推荐**: **t5 / t6 (突发性能实例)**。这类实例有 "CPU 积分" 限制，如果你的 API 请求稍微多一点，CPU 积分耗尽后会被强制降频到 10%，导致服务卡死。**千万别为了省几十块钱买这个。**

### 2.2 数据库白名单
*   购买 RDS 后，默认是**拒绝所有访问**。
*   一定要在 RDS 控制台 -> "白名单与安全组" 中，把你的 ECS **内网IP** 加进去。
*   **不要**把白名单设为 `0.0.0.0/0`，非常危险！

### 2.3 备案流程
1.  购买 ECS (包年包月 3个月以上) 才有备案资格。
2.  进入 [阿里云 ICP 代备案管理系统](https://beian.aliyun.com)。
3.  按照指引拍照上传身份证/营业执照。
4.  阿里云审核 (1天) -> 管局审核 (7-20天)。
5.  **注意**: 即使是这个时候，你也可以先用 IP 进行开发测试，只是小程序暂时发不出去。

## 3. 容量预估
同之前评估：
*   **并发**: 1000+ (ECS u1 2核4G + RDS 基础版)
*   **瓶颈**: 带宽。建议 ECS 购买 **按量付费带宽 (峰值 100M)**，这样每小时只扣几毛钱流量费，但用户打开速度飞快 (比固定 5M 带宽快 20 倍)。
