# 开发与操作手册 (Developer & Ops Manual)

本文档整合了 **后台功能操作** 与 **移动端开发调试** 的相关指南。

---

## 第一部分：管理后台操作指南

### 1. 服务商信息编辑
管理员可在后台编辑设计师、工长、装修公司的详细资料，这些信息会实时同步到 App 端。

#### 1.1 编辑路径
`服务商管理` -> 选择对应列表 (设计师/公司/工长) -> 点击 `编辑`。

#### 1.2 关键字段说明
- **服务区域**: 必须为 JSON 数组格式，如 `["朝阳区", "海淀区"]`。
- **价格设置**:
    - 设计师: `/m²`
    - 工长: `元/天`
- **封面图**: 建议 1200x800px，支持 HTTPS 链接。

#### 1.3 常见问题
- **Q**: 修改后 App 没变？
- **A**: 下拉刷新 App 页面，或重新进入详情页（缓存原因）。

## 第二部分：移动端开发调试指南

### 1. 运行环境
- **Web 模式 (推荐调试 UI)**:
  ```bash
  cd mobile && npm run web
  ```
  访问: `http://localhost:5173`。

- **Android 真机/模拟器**:
  ```bash
  cd mobile && npm run android
  ```

### 2. 接口连接配置
位于 `mobile/src/services/api.ts`。
- **开发环境**: 自动连接 `localhost` 或局域网 IP。
- **生产环境**: 自动连接服务器 IP。

### 3. 常见报错
- **Network Error**: 检查手机和电脑是否在同一 WiFi；检查 `api.ts` 中的 IP 是否配置正确。
- **方形图标**: Web 版调试时的各种图标字体缺失问题属于正常现象，真机上不会出现。
