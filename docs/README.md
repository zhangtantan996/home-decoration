## 🆘 常见问题

### Q1: 文档太多，从哪开始？
**A**: 根据你的角色选择路径：
- **开发者**: CLAUDE_DEV_GUIDE.md → 技术架构设计总览.md → LOCAL_DEV_GUIDE.md
- **运维**: DEPLOYMENT_GUIDE_ZH.md → 全栈部署指南.md
- **测试**: P0_TEST_PLAN.md → BUSINESS_FLOW.md

### Q2: 遇到问题找不到解决方案？
**A**: 按顺序查找：
1. [TROUBLESHOOTING.md](TROUBLESHOOTING.md) - 已知问题
2. [DEPLOYMENT_ISSUES.md](DEPLOYMENT_ISSUES.md) - 部署问题
3. [CLAUDE_DEV_GUIDE.md#问题排查流程](CLAUDE_DEV_GUIDE.md) - 使用排查流程
4. 搜索代码库中的注释

### Q3: 文档内容过时了怎么办?
**A**: 请按照以下流程更新：
```bash
# 1. 编辑文档
vim docs/xxx.md

# 2. 更新"最后更新"时间
最后更新: 2026-01-XX

# 3. 提交 Git
git add docs/xxx.md
git commit -m "docs: 更新 xxx.md - <变更说明>"
```

### Q4: 如何快速搜索文档？
**A**: 使用 grep 搜索关键词：
```bash
# 搜索所有文档
grep -r "React 版本" docs/

# 只搜索标题
grep -r "^##" docs/ | grep "部署"
```
