## 🆘 常见问题

### Q1: 文档太多，从哪开始？
**A**: 根据你的角色选择路径：
- **开发者**: Claude开发参考手册.md → 技术架构设计总览.md → 本地极速开发指南.md
- **运维**: 部署指南.md → 全栈部署指南.md
- **测试**: P0测试计划.md → 业务流程规范.md

### Q2: 遇到问题找不到解决方案？
**A**: 按顺序查找：
1. [问题解决手册.md](问题解决手册.md) - 已知问题
2. [部署问题记录.md](部署问题记录.md) - 部署问题
3. [Claude开发参考手册.md#问题排查流程](Claude开发参考手册.md) - 使用排查流程
4. 搜索代码库中的注释

### Q3: 文档内容过时了怎么办?
**A**: 请按照以下流程更新：
```bash
# 1. 编辑文档
vim docs/xxx.md

# 2. 更新"最后更新"时间
最后更新: 2026-01-XX

# 3. 提交 Git
git add docs/xxx.md
git commit -m "docs: 更新 xxx.md - <变更说明>"
```

### Q4: 如何快速搜索文档？
**A**: 使用 grep 搜索关键词：
```bash
# 搜索所有文档
grep -r "React 版本" docs/

# 只搜索标题
grep -r "^##" docs/ | grep "部署"
```
