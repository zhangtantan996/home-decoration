# UI Memory Index

用途：
- 记录已被认可、值得复用的 UI 方案案例
- 帮助后续页面设计先查相似案例，再决定是否复用

使用规则：
- 这里只做索引，不写长篇方案
- 具体案例放到 `docs/ui-memory/cases/`
- 稳定可复用的通用模式沉淀到 `docs/ui-memory/patterns.md`
- 稳定需要规避的坏模式沉淀到 `docs/ui-memory/anti-patterns.md`
- 新案例默认按 `docs/ui-memory/templates/case-template.md` 填写

索引字段建议：
- 案例名
- 页面类型
- 模块
- 适用场景
- 核心优点
- 主要问题 / 反例点
- 可复用点
- 需避免点
- 对应案例文件
