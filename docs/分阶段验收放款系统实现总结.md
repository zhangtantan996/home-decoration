## 核心功能实现

### 1. 按阶段验收
- 每个阶段独立验收
- 验收通过后进入T+N放款流程
- 支持验收清单模板
- 支持验收备注

### 2. 整体验收一次性放款（核心）
```go
func (s *InspectionService) AcceptAllMilestones(projectID, userID uint64) error {
    return repository.DB.Transaction(func(tx *gorm.DB) error {
        // 1. 检查项目是否存在
        // 2. 检查用户权限
        // 3. 获取所有阶段节点
        // 4. 检查所有阶段是否已提交
        // 5. 一次性验收所有阶段
        // 6. 计算总金额并一次性放款
        // 7. 创建结算单并执行放款
        // 8. 更新项目状态
        // 9. 发送通知
        // 10. 记录审计日志
        return nil
    })
}
```

### 3. 验收清单模板
