# 隐私与数据处理条款

- 版本：`v1.0.0-20260305`
- 生效日期：`2026-03-05`

## 1. 处理目的
平台为完成身份核验、入驻审核、经营管理、风控与合规义务，在必要范围内处理商家数据。

## 2. 数据范围
包括但不限于手机号、联系人信息、证件/执照信息、案例或商品信息、日志与审计信息。

## 3. 安全措施
平台采用传输加密、权限隔离、日志审计、敏感字段保护等措施保障数据安全。

## 4. 数据共享与披露
平台仅在法律法规要求、监管机关要求、司法程序要求或商家授权情况下进行必要披露。

## 5. 数据保留与删除
平台根据法律要求与业务必要性保留数据，到期后按规则删除或匿名化处理。

## 6. 反馈渠道
商家可通过平台客服或运营支持渠道提出数据更正、更新或处理异议申请。
