# 第三方信息共享清单

禾泽云仅在实现具体功能所必需的范围内，向第三方服务商共享必要信息，不会向无关第三方出售个人信息。

## 一、共享原则

- 仅为完成具体业务功能共享必要字段。
- 仅向已实际启用的第三方服务商共享信息。
- 未启用的第三方服务不纳入公开清单，也不应超范围获取数据。
- 平台会要求相关第三方在其职责范围内履行信息保护义务。

## 二、可能涉及的第三方类型

### 1. 短信服务

如启用短信服务，将共享手机号及必要模板标识，用于发送登录、注册、身份核验、商家审核、账户安全验证码或通知。

### 2. 支付服务

如启用支付宝或微信支付，将共享订单金额、订单标识、支付状态处理所需信息，用于完成支付、退款、对账和结果通知。

### 3. 实名核验或企业核验

如启用实名核验或企业核验服务，将共享姓名、身份证号、公司名称、统一社会信用代码、营业执照号等核验所需信息。

### 4. 对象存储服务

如启用对象存储服务，将在图片、资质文件、案例附件、退款凭证、投诉材料等上传与读取场景中处理对应文件数据。

### 5. 地图定位服务

如启用地图定位服务，将在地址选择、服务城市、位置相关能力中处理必要位置信息。

### 6. 即时通信服务

如启用站内消息或即时沟通服务，将在消息同步、会话管理和通知提醒场景中处理必要的会话与消息数据。

## 三、实际启用情况说明

页面展示的第三方共享清单会以平台当前实际启用的服务为准。若某项服务未启用，不应被解释为平台已经向该第三方共享了你的信息。
