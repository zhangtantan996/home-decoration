# 智能报价系统使用说明

## 功能概述

智能报价系统允许用户输入房屋面积、装修风格和所在区域，自动生成半包/全包报价区间、工期预估、材料预算明细和风险项提醒。

## 后端实现

### 数据模型

**QuoteEstimateTemplate** (`server/internal/model/quote_estimate_template.go`)
- 存储不同风格、区域、面积区间的报价模板
- 包含半包/全包价格区间（元/平米）
- 包含工期、材料预算、风险项等信息

### 服务层

**QuoteEstimateService** (`server/internal/service/quote_estimate_service.go`)
- `EstimateQuote()`: 根据面积、风格、区域生成报价
- `GetQuoteTemplate()`: 智能匹配报价模板（支持降级策略）
- `CalculateDuration()`: 根据面积动态计算工期
- `GetMaterialsBudget()`: 根据面积调整材料预算
- `GetRiskItems()`: 获取风险项提醒

### API 接口

**POST /api/v1/quote-estimate**

请求体：
```json
{
  "area": 100,
  "style": "现代简约",
  "region": "一线城市"
}
```

响应：
```json
{
  "code": 200,
  "data": {
    "halfPackMin": 80000,
    "halfPackMax": 120000,
    "fullPackMin": 150000,
    "fullPackMax": 200000,
    "duration": 95,
    "materials": [
      {"category": "水电材料", "budget": 15000},
      {"category": "瓦工材料", "budget": 20000}
    ],
    "riskItems": [
      {"item": "老房拆改", "description": "如果是老房需要拆改，费用另计"}
    ]
  }
}
```

## 小程序端实现

### 页面路径

`mini/src/pages/quote/estimate/index.tsx`

### 功能特性

1. **输入表单**
   - 房屋面积（数字输入）
   - 装修风格（选择器）
   - 所在区域（选择器）

2. **报价结果展示**
   - 半包/全包报价区间卡片
   - 工期预估
   - 材料预算明细列表
   - 风险项提醒（黄色警告框）

3. **交互流程**
   - 用户填写信息 → 点击"生成报价" → 显示结果 → 点击"立即预约"跳转到预约页面

## 报价模板管理

报价模板存储在 `quote_estimate_templates` 表中，可以通过管理后台进行增删改查（未来功能）。

当前模板覆盖：
- 现代简约：一线/二线城市，小/中/大户型
- 北欧：一线/二线城市，中户型
- 中式：一线/二线城市，中户型
- 轻奢：一线/二线城市，中户型
- 日式：一线/二线城市，中户型
- 美式：一线/二线城市，大户型

## 匹配策略

系统采用四级降级匹配策略：

1. **精确匹配**：风格 + 区域 + 面积区间
2. **降级1**：风格 + 面积区间（忽略区域）
3. **降级2**：区域 + 面积区间（忽略风格）
4. **降级3**：仅面积区间

如果所有策略都失败，返回错误提示用户联系客服。

## 工期计算规则

- 基础工期：模板中定义（如90天）
- 面积调整：每增加20平米，工期增加5天
- 上下限：最短30天，最长180天

## 材料预算调整

材料预算以100平米为基准，根据实际面积按比例调整。

例如：
- 基准（100㎡）：水电材料 15000元
- 实际（200㎡）：水电材料 30000元

## 注意事项

1. 报价为系统估算，实际价格以量房后的详细报价为准
2. 风险项提醒需要用户特别注意，可能产生额外费用
3. 报价不包含家具、家电等软装费用
4. 老房拆改、水电改造等费用根据实际情况调整
