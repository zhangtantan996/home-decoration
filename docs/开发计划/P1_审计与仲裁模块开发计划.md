> **优先级**: P1 (高优先级)
> **工作量**: 4人日
> **完成时间**: 第二周
> **依赖模块**: 暂停与争议模块

---

## 一、模块概述

### 1.1 业务目标

完善平台审计与仲裁机制，支持管理员介入争议处理，给出仲裁结论并执行，确保公平公正。

### 1.2 功能范围

1. **审计功能**
   - 管理员查看争议详情
   - 查看双方证据材料
   - 记录审计过程

2. **仲裁功能**
   - 给出仲裁结论
   - 执行仲裁结论（继续施工/退款/其他）
   - 通知双方

### 1.3 业务流程

```
争议提交 → 管理员审计 → 给出仲裁结论 → 执行仲裁 → 通知双方
```

## 三、技术设计

### 3.1 数据模型

#### 3.1.1 项目审计表 (project_audits)

```sql
CREATE TABLE project_audits (
    id BIGSERIAL PRIMARY KEY,
    project_id BIGINT NOT NULL REFERENCES projects(id),
    admin_id BIGINT NOT NULL REFERENCES admins(id),
    audit_type VARCHAR(20) NOT NULL, -- dispute/refund/other
    audit_notes TEXT, -- 审计记录
    conclusion VARCHAR(20), -- continue/refund/partial_refund/other
    conclusion_reason TEXT, -- 仲裁理由
    execution_plan JSONB, -- 执行方案
    status VARCHAR(20) NOT NULL DEFAULT 'pending', -- pending/completed
    created_at TIMESTAMP NOT NULL DEFAULT NOW(),
    completed_at TIMESTAMP,
    updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_project_audits_project_id ON project_audits(project_id);
CREATE INDEX idx_project_audits_admin_id ON project_audits(admin_id);
CREATE INDEX idx_project_audits_status ON project_audits(status);
```

**execution_plan 字段结构**:
```json
{
  "action": "partial_refund",
  "refund_amount": 10000,
  "refund_reason": "施工质量问题，部分退款",
  "continue_construction": false
}
```

### 3.2 后端接口设计

#### 3.2.1 审计接口

**POST /api/v1/admin/projects/:id/audit**

创建审计记录

**请求参数**:
```json
{
  "auditType": "dispute",
  "auditNotes": "已查看双方证据，施工质量确实存在问题"
}
```

**响应**:
```json
{
  "code": 0,
  "message": "审计记录已创建",
  "data": {
    "audit": {
      "id": 789,
      "projectId": 123,
      "status": "pending",
      "createdAt": "2026-03-17T18:00:00Z"
    }
  }
}
```

---

**GET /api/v1/admin/audits**

查询审计列表

**请求参数**:
- `status`: pending/completed
- `page`: 页码
- `pageSize`: 每页数量

**响应**:
```json
{
  "code": 0,
  "data": {
    "list": [
      {
        "id": 789,
        "projectId": 123,
        "auditType": "dispute",
        "status": "pending",
        "createdAt": "2026-03-17T18:00:00Z"
      }
    ],
    "total": 10,
    "page": 1,
    "pageSize": 20
  }
}
```

---

**GET /api/v1/admin/audits/:id**

查询审计详情

**响应**:
```json
{
  "code": 0,
  "data": {
    "audit": {
      "id": 789,
      "project": {...},
      "user": {...},
      "foreman": {...},
      "disputeReason": "施工质量不符合要求",
      "disputeEvidence": [...],
      "auditNotes": "已查看双方证据",
      "status": "pending"
    }
  }
}
```

---

#### 3.2.2 仲裁接口

**POST /api/v1/admin/audits/:id/arbitrate**

提交仲裁结论

**请求参数**:
```json
{
  "conclusion": "partial_refund",
  "conclusionReason": "施工质量问题，部分退款",
  "executionPlan": {
    "action": "partial_refund",
    "refund_amount": 10000,
    "continue_construction": false
  }
}
```

**响应**:
```json
{
  "code": 0,
  "message": "仲裁结论已提交，正在执行",
  "data": {
    "audit": {
      "id": 789,
      "conclusion": "partial_refund",
      "status": "completed",
      "completedAt": "2026-03-17T19:00:00Z"
    }
  }
}
```
