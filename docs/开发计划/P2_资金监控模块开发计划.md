# P2_资金监控模块开发计划

> **优先级**: P2 (中优先级)
> **工作量**: 3人日
> **完成时间**: 第三周
> **依赖模块**: 所有资金流转模块

---

## 一、模块概述

### 1.1 业务目标

完善资金流转监控功能，支持管理员实时查看资金状态、执行资金操作，确保资金安全。

### 1.2 功能范围

1. **资金概览功能**
   - 查看资金总览
   - 查看待放款金额
   - 查看已冻结金额

2. **交易流水功能**
   - 查看交易流水列表
   - 筛选交易类型
   - 导出交易记录

3. **资金操作功能**
   - 冻结资金
   - 解冻资金
   - 手动放款

---

## 三、技术设计

### 3.1 后端接口设计

#### 3.1.1 资金概览接口

**GET /api/v1/admin/finance/overview**

查询资金概览

**响应**:
```json
{
  "code": 0,
  "data": {
    "totalBalance": 1000000,
    "pendingRelease": 500000,
    "frozenAmount": 100000,
    "releasedToday": 50000,
    "statistics": {
      "intentFee": 10000,
      "designFee": 50000,
      "constructionFee": 400000
    }
  }
}
```

---

#### 3.1.2 交易流水接口

**GET /api/v1/admin/finance/transactions**

查询交易流水

**请求参数**:
- `type`: deposit/release/freeze/unfreeze/refund
- `startDate`: 开始日期
- `endDate`: 结束日期
- `page`: 页码
- `pageSize`: 每页数量

**响应**:
```json
{
  "code": 0,
  "data": {
    "list": [
      {
        "id": 1001,
        "type": "deposit",
        "amount": 1000,
        "projectId": 123,
        "userId": 456,
        "createdAt": "2026-03-17T21:00:00Z"
      }
    ],
    "total": 100,
    "page": 1,
    "pageSize": 20
  }
}
```

---

#### 3.1.3 资金操作接口

**POST /api/v1/admin/finance/freeze**

冻结资金

**请求参数**:
```json
{
  "projectId": 123,
  "amount": 10000,
  "reason": "项目暂停，冻结后续款项"
}
```

**响应**:
```json
{
  "code": 0,
  "message": "资金已冻结"
}
```

---

**POST /api/v1/admin/finance/unfreeze**

解冻资金

**请求参数**:
```json
{
  "projectId": 123,
  "amount": 10000,
  "reason": "项目恢复，解冻款项"
}
```

**响应**:
```json
{
  "code": 0,
  "message": "资金已解冻"
}
```

---

**POST /api/v1/admin/finance/manual-release**

手动放款

**请求参数**:
```json
{
  "projectId": 123,
  "milestoneId": 456,
  "amount": 10000,
  "reason": "特殊情况，手动放款"
}
```

**响应**:
```json
{
  "code": 0,
  "message": "放款成功"
}
```
