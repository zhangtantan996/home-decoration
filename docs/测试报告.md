## 三、后端代码编译检查

### ❌ 检查结果：失败（6个错误）

**编译命令**:
```bash
cd server && go build ./cmd/api
```

**错误详情**:

#### 错误1-3：缺少通知方法（`inspection_service.go`）

```
internal/service/inspection_service.go:169:77: not enough arguments in call to dispatcher.NotifyMilestoneSubmitted
internal/service/inspection_service.go:238:15: dispatcher.NotifyMilestoneAccepted undefined
internal/service/inspection_service.go:259:15: dispatcher.NotifyMilestoneRejected undefined
internal/service/inspection_service.go:330:14: dispatcher.NotifyMilestoneRejected undefined
internal/service/inspection_service.go:403:14: dispatcher.NotifyMilestoneResubmitted undefined
```

**问题分析**:
- `NotifyMilestoneSubmitted` 方法参数不匹配（第169行）
- `NotifyMilestoneAccepted` 方法未定义（第238行、第330行）
- `NotifyMilestoneRejected` 方法未定义（第259行）
- `NotifyMilestoneResubmitted` 方法未定义（第403行）

**根本原因**:
`notification_dispatcher.go` 中只实现了以下方法：
- ✅ `NotifyMilestoneSubmitted(ownerUserID, projectID, milestoneID uint64, milestoneName string)` - 已存在
- ✅ `NotifyMilestoneDecision(ownerUserID, providerUserID, projectID, milestoneID uint64, milestoneName string, approved bool, reason string)` - 已存在
- ❌ `NotifyMilestoneAccepted` - **缺失**
- ❌ `NotifyMilestoneRejected` - **缺失**
- ❌ `NotifyMilestoneResubmitted` - **缺失**

#### 错误4：类型转换错误

```
internal/service/inspection_service.go:271:19: cannot use map[string]bool{…}["passed"] (map index expression of type bool) as bool value in argument to dispatcher.NotifyMilestoneDecision
```

**问题分析**:
第271行代码尝试将 `map[string]bool` 的索引结果直接作为 `bool` 参数传递，但Go编译器无法推断类型。

## 六、修复建议

### 🔧 紧急修复：后端编译错误

#### 修复方案1：补充缺失的通知方法

在 `server/internal/service/notification_dispatcher.go` 中添加以下方法：

```go
// NotifyMilestoneAccepted 通知商家节点验收通过
func (d *NotificationDispatcher) NotifyMilestoneAccepted(providerUserID, projectID, milestoneID uint64) {
	if providerUserID == 0 || projectID == 0 || milestoneID == 0 {
		return
	}
	
	var milestone model.Milestone
	if err := repository.DB.First(&milestone, milestoneID).Error; err != nil {
		return
	}
	
	name := strings.TrimSpace(milestone.Name)
	if name == "" {
		name = "当前节点"
	}
	
	_ = d.service.Create(&CreateNotificationInput{
		UserID:      providerUserID,
		UserType:    "provider",
		Title:       "阶段验收已通过",
		Content:     fmt.Sprintf(""%s"验收已通过，款项即将到账。", name),
		Type:        "project.milestone.accepted",
		RelatedID:   milestoneID,
		RelatedType: "milestone",
		ActionURL:   fmt.Sprintf("/projects/%d", projectID),
		Category:    NotificationCategoryProject,
	})
}

// NotifyMilestoneRejected 通知商家节点验收被拒绝
func (d *NotificationDispatcher) NotifyMilestoneRejected(providerUserID, projectID, milestoneID uint64, reason string) {
	if providerUserID == 0 || projectID == 0 || milestoneID == 0 {
		return
	}
	
	var milestone model.Milestone
	if err := repository.DB.First(&milestone, milestoneID).Error; err != nil {
		return
	}
	
	name := strings.TrimSpace(milestone.Name)
	if name == "" {
		name = "当前节点"
	}
	
	content := fmt.Sprintf(""%s"验收未通过，请整改后重新提交。", name)
	if reason != "" {
		content = fmt.Sprintf(""%s"验收未通过。原因：%s", name, reason)
	}
	
	_ = d.service.Create(&CreateNotificationInput{
		UserID:      providerUserID,
		UserType:    "provider",
		Title:       "阶段验收未通过",
		Content:     content,
		Type:        "project.milestone.rejected",
		RelatedID:   milestoneID,
		RelatedType: "milestone",
		ActionURL:   fmt.Sprintf("/projects/%d", projectID),
		Category:    NotificationCategoryProject,
	})
}

// NotifyMilestoneResubmitted 通知用户商家重新提交验收
func (d *NotificationDispatcher) NotifyMilestoneResubmitted(ownerUserID, projectID, milestoneID uint64) {
	if ownerUserID == 0 || projectID == 0 || milestoneID == 0 {
		return
	}
	
	var milestone model.Milestone
	if err := repository.DB.First(&milestone, milestoneID).Error; err != nil {
		return
	}
	
	name := strings.TrimSpace(milestone.Name)
	if name == "" {
		name = "当前节点"
	}
	
	_ = d.service.Create(&CreateNotificationInput{
		UserID:      ownerUserID,
		UserType:    "user",
		Title:       "阶段验收已重新提交",
		Content:     fmt.Sprintf("工长已完成整改并重新提交"%s"验收，请查看。", name),
		Type:        "project.milestone.resubmitted",
		RelatedID:   milestoneID,
		RelatedType: "milestone",
		ActionURL:   fmt.Sprintf("/projects/%d", projectID),
		Category:    NotificationCategoryProject,
	})
}
```
