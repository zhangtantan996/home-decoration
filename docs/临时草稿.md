## 🏠 产品概述

**家装设计一体化平台**是一个连接业主、设计师、施工公司、工长/项目经理（施工团队负责人）和工人的综合性家装服务平台。我们致力于打破传统家装行业的信息壁垒，通过数字化手段为用户提供从设计灵感、服务商选择、项目管理到施工监督的全流程一站式解决方案。

### 对业主

## ✨ 核心功能

### 1. 智能服务商匹配
- **多维筛选**：按地区、类型（设计师/公司/工长）、风格、价格筛选
- **实时定位**：基于地理位置推荐附近优质服务商
- **评价体系**：真实用户评价、案例展示、资质认证

### 2. 设计灵感广场
- **海量案例库**：真实装修案例，按风格、户型、预算分类
- **沉浸式浏览**：高清图片、3D效果图、施工过程记录
- **一键收藏**：保存喜欢的设计，随时查看对比

### 3. 项目全流程管理
- **阶段化管理**：设计阶段 → 施工准备 → 施工进行 → 验收交付
- **里程碑机制**：每个阶段设定明确的交付标准和验收节点
- **任务分解**：将大项目拆分为可管理的小任务
- **进度可视化**：时间线展示，实时了解项目状态

### 4. 资金托管系统
- **安全保障**：第三方托管账户，资金安全有保障
- **分期支付**：按项目里程碑分期释放资金
- **透明流水**：所有交易记录可查，清晰明了
- **纠纷仲裁**：平台介入机制，保护双方权益

### 5. 实时沟通协作
- **即时聊天**：业主与服务商实时沟通，支持文字、图片、语音
- **群组协作**：项目相关方（业主、设计师、工长/项目经理、工人）统一沟通
- **预约管理**：在线预约上门测量、方案讲解、施工验收
- **消息提醒**：重要节点自动提醒，不错过任何关键信息

### 6. 建材商城对接
- **品牌入驻**：知名建材品牌官方授权店铺
- **产品展示**：详细参数、价格、库存信息
- **一键采购**：与项目进度联动，及时采购所需材料

### 7. 评价与信用体系
- **双向评价**：业主评价服务商，服务商评价业主
- **信用积分**：基于历史交易和评价的信用评级
- **黑名单机制**：保护平台生态，过滤不良用户

## 🛠️ 技术架构

### 前端
- **移动端App**：React Native 0.83，支持 iOS/Android/Web 三端
- **管理后台**：React 19 + TypeScript + Ant Design，现代化管理界面

### 后端
- **API服务**：Go 1.21 + Gin 框架，高性能 RESTful API
- **数据库**：PostgreSQL + Redis，数据持久化与缓存
- **实时通信**：WebSocket，支持即时聊天和消息推送

### 部署
- **容器化**：Docker + Docker Compose，一键部署
- **负载均衡**：Nginx 反向代理，支持高并发访问
- **云原生**：支持阿里云、AWS 等主流云平台

## 🚀 商业模式

### 收入来源
1. **服务佣金**：从成交项目中收取一定比例佣金
2. **会员服务**：为服务商提供高级会员功能（优先推荐、数据分析等）
3. **广告收入**：建材品牌广告位、精准营销
4. **增值服务**：第三方服务对接（保险、贷款、监理等）

### 目标用户
- **一线用户**：一二线城市有装修需求的业主（25-45岁）
- **服务商**：独立设计师、设计工作室、装修公司、工长/项目经理（施工团队负责人）
- **建材商**：建材品牌、经销商、建材超市
- **商家中心入口规范**：统一为「我要入驻」，角色选择为设计师入驻（个人/公司）、工长入驻（个人/公司）、装修公司入驻、主材商入驻。

### 入驻资料口径（v1.5.1）
- **设计师（个人/公司）必填**
  - 手机号、验证码、姓名、头像、身份证正反面、身份证号
  - 从业年限（1-50）、服务区域、擅长风格（1-3）、结构化报价（平层/复式/其他）
  - 案例至少 3 套，每套 3-6 张图，每套必须填写案例说明（1-5000）
  - 公司主体额外必填：公司名、执照号、执照图片
- **工长（个人/公司）必填**
  - 手机号、验证码、姓名、头像、身份证正反面、身份证号
  - 施工年限（1-50）、工种（至少1个）、施工亮点（1-3）、服务区域、报价（perSqm）
  - 施工案例至少 3 套，每套 8-12 张图，每套必须填写案例说明（1-5000）
  - 公司主体额外必填：公司名、执照号、执照图片
- **装修公司必填**
  - 手机号、验证码、负责人、头像、公司名、执照号、执照图片
  - 报价必须覆盖全包/半包、服务区域、案例至少 3 套（每套至少 3 图 + 案例说明）
- **主材商必填**
  - 手机号、验证码、店铺名、营业执照号、执照图片
  - 联系人姓名、联系人手机号、营业时间、门店地址
  - 商品 5-20 个，每商品至少 1 图，且必须包含参数与价格

### 入驻条款与留痕口径（v1.5.2）
- 入驻提交前必须勾选并同意：
  - 《平台入驻协议（线上勾选版）》
  - 《平台规则》
  - 《隐私与数据处理条款》
- 入驻接口统一新增：
  - `legalAcceptance.accepted`
  - `legalAcceptance.onboardingAgreementVersion`
  - `legalAcceptance.platformRulesVersion`
  - `legalAcceptance.privacyDataProcessingVersion`
- 申请表落库留痕：
  - `legal_acceptance_json`（版本快照）
  - `legal_accepted_at`（服务端确认时间）
  - `legal_accept_source`（默认 `merchant_web`）
