### 测试2: 验证问题1修复（离线消息显示）
**平台**: Android 真机 + iOS 模拟器

**步骤**:
1. 账号A在线，账号B离线（关闭App）
2. 账号A发送消息给账号B
3. 账号B重新打开App
4. 点击会话进入聊天室
5. **查看控制台日志**

**预期结果**:
- ✅ 会话列表显示对话
- ✅ 点击进入后能看到聊天记录
- ✅ 控制台显示详细日志：
  ```
  [ChatRoom] Subscribing to topic: usrXXX
  [Tinode] 订阅会话: usrXXX limit: 50
  [Tinode] Topic object: exists
  [Tinode] onAllMessagesReceived callback, count: X
  [ChatRoom] getCachedTopicMessages: found X messages
  [ChatRoom] parseTinodeMessages: input count = X
  [ChatRoom] parseTinodeMessages: after filter count = X
  ```

**如果失败**:
- 查看日志中的 `receivedCount` 和 `messageCount` 值
- 查看是否有消息被过滤掉（`All messages filtered out!` 警告）
- 截图日志并报告
