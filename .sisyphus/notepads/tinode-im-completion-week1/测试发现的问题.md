## P0 问题（阻塞性）

### 问题1: 离线消息不显示（Mobile）
- **场景**: 2.6 离线消息
- **平台**: Android 真机 + iOS 模拟器
- **描述**: 会话列表有对话，但点击进入后没有聊天记录显示，只有时间戳
- **严重程度**: P0（阻塞）
- **影响**: 离线消息功能完全不可用

### 问题2: 断线重连后消息不同步（Mobile）
- **场景**: 2.7 断线重连
- **平台**: Android 真机
- **描述**: 
  - 缺少日志: `[Tinode] WebSocket 已连接`
  - 重连后会话列表为空
  - 消息未自动同步
- **严重程度**: P0（阻塞）
- **影响**: 重连功能不完整

### 问题3: 认证失败错误（Mobile）
- **场景**: 未知触发场景
- **平台**: iOS 模拟器
- **错误**: `Failed to subscribe/load messages: CommError:authentication required (401)`
- **描述**: WiFi 显示绿色（已连接），但点击会话进入聊天页面时弹框"加载失败"
- **严重程度**: P0（阻塞）
- **影响**: 无法加载聊天记录

### 问题4: 图片发送失败（Mobile）
- **场景**: 5.2 从相册选择图片
- **平台**: iOS 模拟器 + Android 真机
- **错误**: `TypeError: _e16.set is not a function (it is undefined)`
- **描述**: 选择照片后发送失败，弹窗提示"发送失败"
- **严重程度**: P0（阻塞）
- **影响**: 图片消息功能完全不可用

### 问题5: 相机权限错误（Mobile Android）
- **场景**: 5.3 用相机拍照
- **平台**: Android 真机
- **错误**: `This library does not require Manifest.permission.CAMERA, if you add this permission in manifest then you have to obtain the same.`
- **严重程度**: P0（阻塞）
- **影响**: Android 端无法使用相机拍照

## P1 问题（重要）

### 问题6: 切换会话导致内容消失（Admin）
- **场景**: 3.2 查看会话列表
- **平台**: Admin 管理后台
- **错误**: `Failed to open conversation Error: Tinode history timeout`
- **描述**: 
  - 手动切换会话窗口后会话内容消失
  - 无法正常发送消息
  - 输入框输入后发送，消息不直接显示，需刷新页面
- **严重程度**: P1（重要）
- **影响**: 会话切换体验差

### 问题7: 发送消息后光标丢失（Admin）
- **场景**: 消息发送
- **平台**: Admin 管理后台
- **描述**: 输入框输入消息，点击回车后消息发送成功，但光标不在输入框了
- **严重程度**: P1（重要）
- **影响**: 用户体验差，需要重新点击输入框

### 问题8: iOS 输入框无法弹出键盘（Mobile）
- **场景**: 聊天输入
- **平台**: iOS 模拟器
- **描述**: 点击输入框无法主动弹出输入法
- **严重程度**: P1（重要）
- **影响**: iOS 端无法输入消息

### 问题9: Android 键盘收起后布局错位（Mobile）
- **场景**: 聊天输入
- **平台**: Android 真机
- **描述**: 
  - 点击输入框可以弹出输入法
  - 点击其他地方输入法消失
  - 但输入框没有回到原位置，比原来高了一部分，下方留了空间
- **严重程度**: P1（重要）
- **影响**: 布局错位，用户体验差
