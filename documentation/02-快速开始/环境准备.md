# 环境准备

在开始开发之前，请确保您的本地机器已安装并配置好以下环境。本项目采用单体仓库（Monorepo）结构，包含 Go 后端、React 管理后台、React Native 移动端以及微信小程序。

**最后更新**: 2026-01-25

---

## 核心环境要求

### 必需工具
| 工具 | 最低版本 | 用途 |
|------|---------|------|
| **Node.js** | 20.x | 前端（Admin, Mobile, Mini）构建与运行 |
| **Go** | 1.21 | 后端服务开发与运行 |
| **Docker** | 最新版 | 运行数据库 (PostgreSQL) 和缓存 (Redis) |
| **Git** | 2.x | 版本控制与协作 |

### 推荐工具
- **Air**: Go 后端热重载工具，提升开发效率。
- **Make**: 用于执行项目中的 Makefile 命令（主要在 `server/` 目录下）。
- **WeChat DevTools**: 微信小程序开发与调试。
- **Android Studio / Xcode**: 移动端（React Native）开发与打包。

## 安装指南

### macOS

推荐使用 [Homebrew](https://brew.sh/) 进行安装：

```bash
# 安装 Node.js (推荐使用 nvm 管理版本)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
nvm install 20
nvm use 20

# 安装 Go 1.21
brew install go@1.21

# 安装 Docker Desktop
brew install --cask docker

# 安装 Air (Go 热重载)
go install github.com/cosmtrek/air@latest
```

### Linux (Ubuntu/Debian)

```bash
# Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# Go
sudo apt-get install -y golang-1.21-go

# Docker
sudo apt-get update
sudo apt-get install ca-certificates curl gnupg
# (请参考 Docker 官方文档完成后续步骤)

# Air
go install github.com/cosmtrek/air@latest
```

### Windows

推荐使用 [Chocolatey](https://chocolatey.org/) 或直接下载安装包：

```powershell
# Node.js
choco install nodejs-lts --version=20.0.0

# Go
choco install golang --version=1.21.0

# Docker
choco install docker-desktop

# Air
go install github.com/cosmtrek/air@latest
```

## 环境验证

安装完成后，请在终端执行以下命令验证安装是否成功：

```bash
# 验证 Node.js
node --version  # 应显示 v20.x.x
npm --version

# 验证 Go
go version      # 应显示 go version go1.21.x ...

# 验证 Docker
docker --version
docker compose version

# 验证 Git
git --version
```

## 常见问题

### 1. Node.js 版本冲突
Admin 管理后台锁定使用 `React 18.3.1`，而 Mobile 移动端使用 `React 19.2.0`。虽然它们运行在不同的目录下，但建议统一使用 **Node 20+** 以获得最佳兼容性。

### 2. Docker 资源分配
建议为 Docker Desktop 分配至少 **4GB RAM**，以确保 PostgreSQL 和 Redis 能够稳定运行。

### 3. Go 代理设置 (国内用户)
如果您在中国大陆开发，建议设置 Go 代理以加速依赖下载：
```bash
go env -w GOPROXY=https://goproxy.cn,direct
```
