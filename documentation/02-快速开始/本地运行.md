# 本地运行

**最后更新**: 2026-01-25

## 快速启动（推荐）

本项目支持使用 Docker 一键启动基础设施，或在根目录使用统一脚本启动各端。

### 1. 启动基础设施
在项目根目录下执行以下命令，启动 PostgreSQL 和 Redis：

```bash
npm run infra
# 或者直接使用 docker-compose
docker compose up -d db redis
```

### 2. 启动后端 API
```bash
cd server
make dev
```
> **注**：后端集成 `air` 工具支持热重载，代码修改保存后会自动重新编译并重启服务。

### 3. 启动管理后台 (Admin)
```bash
cd admin
npm install
npm run dev
```

### 4. 启动移动端 (Mobile)
```bash
cd mobile
npm install
npm run start
```
> **注**：移动端使用 React Native 0.83，可以通过 Metro 控制台选择在 iOS/Android 模拟器或浏览器中运行。

## 常见问题

1. **首次运行缓慢？**
   前端项目安装依赖时受网络环境影响可能较慢，建议配置镜像源：
   ```bash
   npm config set registry https://registry.npmmirror.com
   ```

2. **数据库连接失败？**
   确保已运行 `npm run infra`。若端口冲突，请检查本地是否有其他服务占用 5432 (DB) 或 6380 (Redis) 端口。

3. **端口占用？**
   - API: 8080
   - Admin: 5173
   - Mobile: 5174
