# 功能模块

**最后更新**: 2026-01-25

## 1. 功能总览

本平台旨在构建透明、可控、省心的装修生态系统，连接业主、设计师、工长及工人。以下为核心功能模块的开发状态与优先级。

### 核心引擎（P0）
| 功能模块 | 说明 | 状态 | 优先级 |
|---------|------|------|--------|
| 用户认证 | 多角色登录/注册、短信验证码、JWT鉴权 | ✅ 已完成 | P0 |
| 项目发布 | 业主发布装修需求、预算及户型信息 | ✅ 已完成 | P0 |
| 即时通讯 | Tinode 聊天系统（支持单聊、群聊、消息推送） | 🚧 开发中 (85%) | P0 |
| 资金托管 | 节点化资金预存、滑动支付确认、安全隔离 | ✅ 已完成 | P0 |
| 匹配算法 | 基于 LBS 和维度权重（还原度、预算控制）推荐服务商 | ✅ 已完成 | P0 |
