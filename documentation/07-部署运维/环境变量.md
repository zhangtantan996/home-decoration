# 环境变量

本文档列出了“装修设计一体化平台”中使用的所有环境变量及其说明。这些变量通常在 `.env` 文件中定义，或在部署环境（如 Docker, Kubernetes）中直接设置。

**最后更新**: 2026-01-25

## 基础配置

| 变量名 | 必填 | 默认值 | 说明 |
| :--- | :---: | :--- | :--- |
| `APP_ENV` | 否 | `local` | 应用运行环境。可选值：`local`, `docker`, `production`。 |
| `SERVER_HOST` | 否 | `0.0.0.0` | 后端服务监听地址。 |
| `SERVER_PORT` | 否 | `8080` | 后端服务监听端口。 |
| `SERVER_MODE` | 否 | `debug` | Gin 框架运行模式。生产环境建议设置为 `release`。 |
| `SERVER_PUBLIC_URL` | 否 | `http://localhost:8080` | 服务的外部访问基础路径（用于生成链接等）。 |
| `CORS_ALLOWED_ORIGINS` | 否 | `*` | 允许跨域的源列表，多个值用逗号分隔。 |

## 数据库配置 (PostgreSQL)

| 变量名 | 必填 | 默认值 | 说明 |
| :--- | :---: | :--- | :--- |
| `DATABASE_HOST` | 是 | `localhost` | 数据库主机地址。 |
| `DATABASE_PORT` | 否 | `5432` | 数据库端口。 |
| `DATABASE_USER` | 是 | - | 数据库连接用户名。 |
| `DATABASE_PASSWORD` | 是 | - | 数据库连接密码。 |
| `DATABASE_DBNAME` | 是 | `home_decoration` | 数据库名称（推荐使用，映射到配置项 `database.dbname`）。 |
| `DATABASE_NAME` | 否 | - | 兼容旧变量名（等价于 `DATABASE_DBNAME`，二选一即可）。 |
| `DATABASE_SSLMODE` | 否 | `disable` | SSL 连接模式。可选值：`disable`, `require`, `verify-ca`, `verify-full`。 |
| `DATABASE_AUTO_MIGRATE` | 否 | `false` | 是否在启动时自动运行数据库迁移。 |

## 缓存配置 (Redis)

| 变量名 | 必填 | 默认值 | 说明 |
| :--- | :---: | :--- | :--- |
| `REDIS_HOST` | 否 | `localhost` | Redis 主机地址。 |
| `REDIS_PORT` | 否 | `6379` | Redis 端口。 |
| `REDIS_PASSWORD` | 否 | - | Redis 连接密码（若无则留空）。 |
| `REDIS_DB` | 否 | `0` | Redis 数据库索引。 |

## 安全与认证

| 变量名 | 必填 | 默认值 | 说明 |
| :--- | :---: | :--- | :--- |
| `JWT_SECRET` | 是 | - | 用于 JWT 签名的密钥。**生产环境必须修改且保持高度机密**。 |
| `JWT_EXPIRE_HOUR` | 否 | `72` | JWT Token 的过期时间（小时）。 |
| `ENCRYPTION_KEY` | 是 | - | 32 字节密钥，用于 AES-256 加密敏感数据（如身份证、银行卡号）。 |

## 日志配置

| 变量名 | 必填 | 默认值 | 说明 |
| :--- | :---: | :--- | :--- |
| `LOG_LEVEL` | 否 | `info` | 日志记录级别。可选：`debug`, `info`, `warn`, `error`。 |
| `LOG_FILE` | 否 | `logs/backend.log` | 日志文件输出路径。 |

## 第三方服务配置 (可选)

### 阿里云短信 (SMS)
| 变量名 | 必填 | 说明 |
| :--- | :---: | :--- |
| `SMS_PROVIDER` | 否 | 短信服务提供商。可选：`mock`（开发测试） / `aliyun`（生产推荐）。 |
| `SMS_ACCESS_KEY_ID` | 否 | 阿里云 AccessKey ID。 |
| `SMS_ACCESS_KEY_SECRET` | 否 | 阿里云 AccessKey Secret。 |
| `SMS_SIGN_NAME` | 否 | 短信签名名称。 |
| `SMS_TEMPLATE_CODE` | 否 | 短信模板代码。 |

### 阿里云对象存储 (OSS)
| 变量名 | 必填 | 说明 |
| :--- | :---: | :--- |
| `OSS_ENDPOINT` | 否 | OSS 地域节点（如 `oss-cn-hangzhou.aliyuncs.com`）。 |
| `OSS_ACCESS_KEY_ID` | 否 | 阿里云 AccessKey ID。 |
| `OSS_ACCESS_KEY_SECRET` | 否 | 阿里云 AccessKey Secret。 |
| `OSS_BUCKET_NAME` | 否 | OSS Bucket 名称。 |

### 地图服务 (高德)
| 变量名 | 必填 | 说明 |
| :--- | :---: | :--- |
| `AMAP_API_KEY` | 否 | 高德地图 Web 服务 API Key。 |

### 微信小程序
| 变量名 | 必填 | 说明 |
| :--- | :---: | :--- |
| `WECHAT_MINI_APPID` | 否 | 微信小程序 AppID。 |
| `WECHAT_MINI_SECRET` | 否 | 微信小程序 AppSecret。 |

## Tinode IM 配置

| 变量名 | 必填 | 说明 |
| :--- | :---: | :--- |
| `TINODE_DATABASE_DSN` | 是 | Tinode 服务的数据库连接字符串。 |
| `TINODE_UID_ENCRYPTION_KEY` | 是 | 用于加密 Tinode 用户 ID 的密钥（16字节 Base64）。 |
| `TINODE_AUTH_TOKEN_KEY` | 是 | 用于生成 Tinode 认证 Token 的密钥（32字节 Base64）。 |
| `TINODE_API_KEY_SALT` | 是 | Tinode API Key 盐值（32字节 Base64）。 |
| `TINODE_API_KEY` | 是 | 客户端访问 Tinode 使用的 API Key。 |
| `TINODE_SERVER_URL` | 否 | Tinode 服务器地址（移动端使用，默认为 `http://localhost:6060`）。 |

## 前端与工具

| 变量名 | 必填 | 默认值 | 说明 |
| :--- | :---: | :--- | :--- |
| `VITE_API_URL` | 否 | `http://localhost:8080/api/v1` | 管理后台前端请求后端的 API 基础地址。 |
| `TARO_APP_API_BASE` | 否 | `http://localhost:8080/api/v1` | 微信小程序请求后端的 API 基础地址。 |

## 配置示例 (`.env`)

```bash
# 应用环境
APP_ENV=production

# 服务配置
SERVER_PORT=8080
SERVER_MODE=release

# 数据库
DATABASE_HOST=db.example.com
DATABASE_USER=admin
DATABASE_PASSWORD=secure_password
DATABASE_DBNAME=home_decoration

# 安全密钥 (示例值，请勿直接使用)
JWT_SECRET=zX9b...r2Pq
ENCRYPTION_KEY=6f5d...a2e1
```

> **注意**: 在生产环境中，请确保 `.env` 文件权限正确（如 600），且不要将其提交到代码仓库。
